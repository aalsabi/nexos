# NexOS - Linux Distribution

## ما هي NexOS؟

توزيعة لينكس حديثة مبنية على **Debian sid**، موجهة للمستخدمين المنتقلين من Windows و macOS.
تجمع بين استقرار قاعدة Debian وتحديثات sid مع واجهة مستخدم عصرية، وتركيز خاص على الألعاب
ودعم المطورين ودعم العربية.

## المميزات الرئيسية

- **قاعدة Debian sid** – أحدث الحزم مع صلابة Debian.
- **GNOME Wayland فقط** – أداء أفضل وأمان أعلى.
- **أدوات نظام مكتوبة بـ C#/GTK4/libadwaita** – تجربة متسقة وحديثة.
- **مكدس ألعاب كامل** – Steam, Proton, Wine, Lutris, Heroic, Gamemode, MangoHud.
- **دعم العربية الكامل** – RTL, خطوط, تعريب واجهات.
- **تطبيقات Flatpak** – عزل وتحديثات سريعة.

## لمن NexOS؟

- مستخدمين قادمين من Windows أو macOS.
- مطورين يحتاجون بيئة حديثة (.NET, containers, IDEs).
- لاعبين يريدون تجربة ألعاب قوية على لينكس.
- محبي التصميم النظيف والواجهات المتسقة.

## البناء من المصدر

```bash
git clone https://github.com/GITHUB_USERNAME/nexos-project
cd nexos-project
./scripts/build-all.sh
```
